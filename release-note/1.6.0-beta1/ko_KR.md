## [1.6.0-beta1]

### New
- 저장 장치에 대한 사용자 지정 표시 기능이 추가되었습니다
- USB 클래스 저장 장치 관리 기능이 추가되었습니다
- RAID 메타데이터의 안정적인 저장 메커니즘이 추가되어, 시스템 재설치 또는 장치 교체 후 원래 배열을 자동으로 다시 인식하고 마운트할 수 있습니다
- 장치 이미지 및 로그인 배경화면의 사용자 지정 표시 기능이 추가되었습니다

### Fixed
- mod_management 모듈의 pull 제한 문제를 수정했습니다
- SMB가 클라이언트로 동작할 때 발생하던 연결 제한 문제를 수정했습니다
- SMB가 예상대로 "공유 권한이 없는 폴더를 건너뛰지 못하던" 문제를 수정했습니다
- smartd 서비스의 간헐적 깨우기로 인해 디스크가 정상 절전 상태에 들어가지 못하던 문제를 수정했습니다
- 비정상적인 docker0 네트워크 인터페이스로 인해 Docker 시작이 실패하던 문제를 수정했습니다
- 데이터 테이블 누락 또는 테이블 구조 이상으로 인해 Files 서비스 시작이 실패하던 문제를 수정했습니다

### Optimized
- 네트워크 카드 정보 표시를 최적화했습니다
- 설정 페이지 표시를 최적화했습니다
- ZimaOS에서 로그인되지 않은 상태의 요청 목록 처리 방식을 최적화했습니다
- 애플리케이션 pull 실패 시 오류 안내 문구를 최적화했습니다
- 공식 ZimaCube 장치의 일곱 번째 드라이브 표시 순서를 조정했습니다
