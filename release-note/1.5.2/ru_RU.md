## [1.5.2]
### New
- Добавлена ​​поддержка устройств ИБП по протоколу USB.
- Добавлен предварительный просмотр Live Photo и автоматическое воспроизведение изображений HEIC в файлах.
- Добавлена ​​функция очистки кэша приложений в один клик.
- Добавлен встроенный редактор Markdown.

### Fixed
- Исправлено ненормальное отображение «ZimaOS-HD» в файлах, когда USB-накопитель используется в качестве системного диска.
- Исправлена ​​ошибка, из-за которой zima-хранилище не открывалось в Бэкапе при загрузке с USB-накопителя.
- Исправлена ​​проблема, из-за которой устройства хранения не отображались в файлах из-за сбоя анализа типа диска.
- Исправлены аномалии отображения, когда пути к точкам монтирования содержат пробелы.
- Исправлен неправильный расчет размера при переносе каталогов, содержащих пробелы.
- Исправлены конфликты имен проектов при импорте проектов с помощью docker-cli.
- Исправлена ​​проблема со стабильностью, приводившая к неожиданному завершению работы службы avahi-daemon.
- Исправлена ​​критическая ошибка, из-за которой разделы mmcblkboot неправильно отображались в качестве выбираемых устройств при создании хранилища (это загрузочные разделы, зарезервированные системой, и их никогда нельзя форматировать).
- Исправлено неточное поведение «Пропустить» в файлах во время разрешения конфликта вырезания/вставки файлов.
- Исправлена ​​отсутствующая поддержка кодовой страницы CP850 в Samba (приводившая к искажению имен файлов на клиентах Windows).

### Optimized
- Улучшены сообщения по умолчанию после импорта контейнеров с помощью запуска Docker.
- Оптимизированная логика индекса LocalStorage.
- Оптимизирован первоначальный интерфейс модуля резервного копирования и интерфейс управления задачами.
- Уточненные описания политик резервного копирования для большей точности и ясности.
- Масштаб сетевого виджета теперь динамически настраивается.
- Оптимизированная логика рекомендаций магазина приложений.
- Расширенный диапазон выбираемых источников резервного копирования.
- Уточнена формулировка стратегии резервного копирования для лучшей читаемости.
