## [1.5.3-beta1]

### Fixed
- UPS 시작 실패 수정
- 특정 백업 데이터의 비정상적인 표시 수정
- 아바타를 변경할 수 없는 문제가 해결되었습니다.

### Optimized
- App Store 추천 섹션의 최적화된 표시 로직

- 최적화된 승인 논리: 시간 초과 제어 추가, IPv4 대체 요청 추가, 승인 가용성 향상
