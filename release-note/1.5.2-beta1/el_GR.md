## [1.5.2-beta1]
### New
- Προστέθηκε υποστήριξη για συσκευές UPS μέσω πρωτοκόλλου USB
- Προστέθηκε προεπισκόπηση Live Photo και αυτόματη αναπαραγωγή για εικόνες HEIC στα Αρχεία
- Προστέθηκε η λειτουργία εκκαθάρισης της προσωρινής μνήμης εφαρμογών με ένα κλικ
- Προστέθηκε ενσωματωμένο πρόγραμμα επεξεργασίας Markdown

### Fixed
- Διορθώθηκε η μη κανονική εμφάνιση του "ZimaOS-HD" στα Αρχεία όταν χρησιμοποιείται μονάδα USB ως δίσκος συστήματος
- Διορθώθηκε η αδυναμία ανοίγματος zima-storage στη σελίδα Backup όταν χρησιμοποιείται μονάδα USB ως δίσκος συστήματος
- Διορθώθηκε το πρόβλημα όπου οι συσκευές αποθήκευσης δεν εμφανίζονταν στα Αρχεία λόγω αποτυχίας ανάλυσης τύπου δίσκου
- Διορθώθηκαν ανωμαλίες οθόνης όταν οι διαδρομές σημείου προσάρτησης περιέχουν κενά
- Διορθώθηκε ο μη φυσιολογικός υπολογισμός μεγέθους κατά τη μετεγκατάσταση καταλόγων που περιέχουν κενά
- Διορθώθηκαν οι διενέξεις ονομάτων έργου κατά την εισαγωγή έργων με χρήση docker-cli
- Διορθώθηκε το πρόβλημα σταθερότητας που προκάλεσε απροσδόκητο τερματισμό της υπηρεσίας avahi-daemon

### Optimized
- Βελτιωμένα προεπιλεγμένα μηνύματα προτροπής μετά την εισαγωγή κοντέινερ μέσω docker run
- Βελτιστοποιημένη λογική LocalStorage Index
- Βελτιστοποιήθηκε η διεπαφή της αρχικής μονάδας δημιουργίας αντιγράφων ασφαλείας και η διεπαφή διαχείρισης εργασιών
- Βελτιωμένες περιγραφές πολιτικής δημιουργίας αντιγράφων ασφαλείας για μεγαλύτερη ακρίβεια και σαφήνεια
- Η κλίμακα του γραφικού στοιχείου δικτύου προσαρμόζεται τώρα δυναμικά
- Βελτιστοποιημένη λογική πρότασης καταστήματος εφαρμογών
