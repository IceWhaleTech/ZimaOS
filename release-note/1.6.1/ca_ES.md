## [1.6.1]

### Fixed
- S'ha corregit el problema pel qual els dispositius USB no es muntaven automàticament després d'una actualització.
- S'ha corregit el problema pel qual les tasques de còpia de seguretat continuaven fent còpies al punt de muntatge del sistema després de perdre's el punt de muntatge.
- S'ha corregit el problema pel qual els mòduls mod no iniciaven els serveis d'acord amb la política de servei després de reiniciar.
- S'ha corregit el problema pel qual el nom del punt de muntatge no coincidia amb el nom real del dispositiu quan aquest contenia espais.

### Optimized
- S'ha optimitzat la lògica de visualització de la safata de discs quan un dispositiu de tercers té massa discs.
- S'han afegit les traduccions que faltaven per a algunes funcions de visualització d'emmagatzematge.
