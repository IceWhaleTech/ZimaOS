## [1.6.1]

### Fixed
- Fixed the issue where USB devices would not auto-mount after an upgrade.
- Fixed the issue where backup tasks would still back up to the system mount point after the mount point was lost.
- Fixed the issue where mod modules would not start services according to the service policy after a reboot.
- Fixed the issue where the mount point name would be inconsistent with the actual device name when the device name contained spaces.

### Optimized
- Optimized the disk shelf display logic when a third-party device has too many disks.
- Added missing translations for some storage display features.
