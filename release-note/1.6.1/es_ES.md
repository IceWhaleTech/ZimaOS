## [1.6.1]

### Fixed
- Se corrigió el problema por el que los dispositivos USB no se montaban automáticamente después de una actualización.
- Se corrigió el problema por el que las tareas de copia de seguridad seguían realizando copias en el punto de montaje del sistema después de que se perdiera el punto de montaje.
- Se corrigió el problema por el que los módulos mod no iniciaban servicios según la política de servicio después de reiniciar.
- Se corrigió el problema por el que el nombre del punto de montaje no coincidía con el nombre real del dispositivo cuando el nombre del dispositivo contenía espacios.

### Optimized
- Se optimizó la lógica de visualización de la bandeja de discos cuando un dispositivo de terceros tiene demasiados discos.
- Se añadieron las traducciones que faltaban para algunas funciones de visualización del almacenamiento.
