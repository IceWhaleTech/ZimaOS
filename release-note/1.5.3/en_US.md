## [1.5.3]

### New
- Added Realtek r8152 network card driver
- Added driver support for Intel Battlemage Series GPUs

### Fixed
- Fixed UPS startup failure
- Fixed abnormal display of certain backup data
- Fixed issue where avatar could not be changed
- Fixed Midnight Commander default configuration path issue
- Fixed issue preventing BTOP from opening on some devices
- Fixed issue where certain NVIDIA graphics cards could not be selected in app settings

### Optimized
- Optimized display logic of the App Store recommendation section
- Optimized Avahi service configuration to reduce the chance of being terminated by other apps
- Optimized authorization logic: added timeout control, added IPv4 fallback requests, improved authorization availability
- Fixed white screen issue when opening popups in Settings
