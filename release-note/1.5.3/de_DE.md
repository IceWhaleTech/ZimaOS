## [1.5.3]

### New
- Realtek r8152-Netzwerkkartentreiber hinzugefügt
- Treiberunterstützung für GPUs der Intel Battlemage-Serie hinzugefügt

### Fixed
- USV-Startfehler behoben
- Die abnormale Anzeige bestimmter Sicherungsdaten wurde behoben
- Problem behoben, bei dem der Avatar nicht geändert werden konnte
- Problem mit dem Standardkonfigurationspfad von Midnight Commander behoben
- Problem behoben, das das Öffnen von BTOP auf einigen Geräten verhinderte
- Problem behoben, bei dem bestimmte NVIDIA-Grafikkarten in den App-Einstellungen nicht ausgewählt werden konnten

### Optimized
- Optimierte Anzeigelogik des App Store-Empfehlungsbereichs
- Optimierte Avahi-Dienstkonfiguration, um die Wahrscheinlichkeit zu verringern, dass sie von anderen Apps beendet wird
- Optimierte Autorisierungslogik: Zeitüberschreitungskontrolle hinzugefügt, IPv4-Fallback-Anfragen hinzugefügt, verbesserte Autorisierungsverfügbarkeit
- Das Problem mit dem weißen Bildschirm beim Öffnen von Popups in den Einstellungen wurde behoben
