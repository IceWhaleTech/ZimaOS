## [1.5.3]

### New
- Se agregó el controlador de la tarjeta de red Realtek r8152.
- Se agregó compatibilidad con controladores para las GPU de la serie Intel Battlemage

### Fixed
- Se corrigió el error de inicio del UPS
- Fixed abnormal display of certain backup data
- Se solucionó el problema por el cual no se podía cambiar el avatar.
- Se solucionó el problema de la ruta de configuración predeterminada de Midnight Commander.
- Se solucionó el problema que impedía que BTOP se abriera en algunos dispositivos
- Se solucionó el problema por el cual ciertas tarjetas gráficas NVIDIA no se podían seleccionar en la configuración de la aplicación.

### Optimized
- Lógica de visualización optimizada de la sección de recomendaciones de la App Store
- Configuración optimizada del servicio Avahi para reducir la posibilidad de que otras aplicaciones lo cancelen.
- Lógica de autorización optimizada: control de tiempo de espera agregado, solicitudes de respaldo de IPv4 agregadas, disponibilidad de autorización mejorada
- Se solucionó el problema de la pantalla blanca al abrir ventanas emergentes en Configuración
