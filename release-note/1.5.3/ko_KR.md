## [1.5.3]

### New
- Realtek r8152 네트워크 카드 드라이버가 추가되었습니다.
- Intel Battlemage 시리즈 GPU에 대한 드라이버 지원이 추가되었습니다.

### Fixed
- UPS 시작 실패 수정
- 특정 백업 데이터의 비정상적인 표시 수정
- 아바타를 변경할 수 없는 문제가 해결되었습니다.
- Midnight Commander 기본 구성 경로 문제 수정
- 일부 장치에서 BTOP가 열리지 않는 문제가 해결되었습니다.
- 앱 설정에서 특정 NVIDIA 그래픽 카드를 선택할 수 없는 문제가 해결되었습니다.

### Optimized
- App Store 추천 섹션의 최적화된 표시 로직
- 다른 앱에 의해 종료될 가능성을 줄이기 위해 Avahi 서비스 구성을 최적화했습니다.
- 최적화된 승인 논리: 시간 초과 제어 추가, IPv4 대체 요청 추가, 승인 가용성 향상
- 설정에서 팝업을 열 때 흰색 화면 문제가 해결되었습니다.
