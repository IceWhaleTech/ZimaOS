## [1.6.0-beta2]

### Fixed
- കൃത്യമായ ഡിസ്ക് സൂചിക സാധ്യതാപരമായി ലഭിക്കാത്ത പ്രശ്നം പരിഹരിച്ചു, WebUI-യിൽ ഇടയ്ക്കിടെ അപ്രത്യക്ഷമാകുന്ന ഡിസ്കുകളുടെ പ്രശ്നം പരിഹരിച്ചു.
- പാർട്ടീഷനുകളുള്ള USB ഉപകരണങ്ങൾ സ്വയമേവ മൌണ്ട് ചെയ്യാൻ കഴിയാത്ത പ്രശ്നം പരിഹരിച്ചു.
- സുതാര്യമായ പശ്ചാത്തലമുള്ള ഇഷ്‌ടാനുസൃത ഉപകരണ ചിത്രങ്ങൾ കറുപ്പായി പ്രദർശിപ്പിച്ച പ്രശ്‌നം പരിഹരിച്ചു.

### Improved
- ലോജിക് പ്രവർത്തനക്ഷമമാക്കുന്ന USB ഉപകരണം ഒപ്റ്റിമൈസ് ചെയ്തു; ഇത് ഇപ്പോൾ പ്രവർത്തനക്ഷമമാക്കുമ്പോഴും ഫ്ലെക്സിബിലിറ്റി മെച്ചപ്പെടുത്തുമ്പോഴും ഒരു മൌണ്ട് പോയിൻ്റ് തിരഞ്ഞെടുക്കുന്നതിനെ പിന്തുണയ്ക്കുന്നു.
- കണ്ടെത്തുന്നത് എളുപ്പമാക്കുന്നതിന് സ്റ്റോറേജ് പ്രവർത്തനക്ഷമ ബട്ടണിൻ്റെ ഡിസ്പ്ലേ ലോജിക് ഒപ്റ്റിമൈസ് ചെയ്തു.




## [1.6.0-beta1]

### New
- സംഭരണ ഉപകരണങ്ങൾക്കായി ഇഷ്ടാനുസൃത പ്രദർശനം ചേർത്തു
- USB ക്ലാസ് സംഭരണ ഉപകരണ മാനേജ്മെന്റ് ചേർത്തു
- RAID മെറ്റാഡാറ്റയ്ക്ക് വിശ്വസനീയമായ സേവിംഗ് സംവിധാനം ചേർത്തു; സിസ്റ്റം വീണ്ടും ഇൻസ്റ്റാൾ ചെയ്യുകയോ ഉപകരണം മാറ്റുകയോ ചെയ്ത ശേഷം പ്രാഥമിക അറേ സ്വയം വീണ്ടും തിരിച്ചറിയുകയും മൗണ്ട് ചെയ്യുകയും ചെയ്യുന്നു
- ഉപകരണ ചിത്രങ്ങൾക്കും ലോഗിൻ വാൾപേപ്പറിനും ഇഷ്ടാനുസൃത പ്രദർശനം ചേർത്തു

### Fixed
- mod_management മോഡ്യൂളിലെ pull നിയന്ത്രണ പ്രശ്നം പരിഹരിച്ചു
- SMB ക്ലയന്റായി പ്രവർത്തിക്കുമ്പോഴുള്ള കണക്ഷൻ നിയന്ത്രണ പ്രശ്നം പരിഹരിച്ചു
- SMB പ്രതീക്ഷിച്ചതുപോലെ "പങ്കിടൽ അനുമതി ഇല്ലാത്ത ഫോൾഡറുകൾ ഒഴിവാക്കാത്ത" പ്രശ്നം പരിഹരിച്ചു
- smartd സേവനത്തിന്റെ ഇടയ്ക്കിടെയുള്ള wake-up കാരണം ഡിസ്കുകൾ സാധാരണ sleep നിലയിലേക്ക് പ്രവേശിക്കാതിരുന്ന പ്രശ്നം പരിഹരിച്ചു
- docker0 നെറ്റ്‌വർക്ക് ഇന്റർഫേസ് അസാധാരണമായതിനാൽ ഉണ്ടായ Docker സ്റ്റാർട്ടപ്പ് പരാജയം പരിഹരിച്ചു
- ഡാറ്റ പട്ടികകൾ ഇല്ലാതെയോ പട്ടിക ഘടന അസാധാരണമായതുകൊണ്ടോ ഉണ്ടായ Files സേവന സ്റ്റാർട്ടപ്പ് പരാജയം പരിഹരിച്ചു
