## [1.5.4-alpha1]

### New
- **Teste alfa: Adicionado recurso de pasta criptografada (esse recurso está atualmente em fase de teste, faça backup de seus dados antes de usá-lo)**
- Gerenciamento aprimorado de SMB: função de compartilhamento de parada adicionada; após parar, os itens compartilhados permanecem na lista para facilitar o gerenciamento
- Adicionado botão Armazenamento na barra lateral Arquivos para facilitar o gerenciamento do armazenamento montado

### Fixed
- Corrigido problema em que o processo de reparo do RAID não conseguia selecionar os discos corretamente
- Corrigido problema em que os membros não podiam criar usuários com o nome "elif"
- Corrigida falha de análise na rede quando uma placa de rede possui vários nomes lógicos
- Corrigida falha de download em Arquivos quando nomes de arquivos contêm vírgulas (,)
- Corrigido o problema em que o cancelamento do compartilhamento SMB de uma pasta também cancelava o compartilhamento de subpastas
- Corrigida falha de visualização de alguns arquivos HEIC em Arquivos
- Corrigido problema com arquivos de soquete e pipe em diretórios de backup

### Optimized
- Página de gerenciamento de armazenamento otimizada
- Solicitações de erro otimizadas para backups
- Exibição otimizada da lista de arquivos em Arquivos: ultrapassou o limite anterior de 10.000 navegação de arquivos, agora suporta a navegação de mais arquivos normalmente
