## [1.5.4-alpha1]

### New
- **Альфа-тест: добавлена ​​функция зашифрованных папок (эта функция в настоящее время находится на этапе тестирования, перед ее использованием создайте резервную копию данных)**
- Улучшено управление SMB: добавлена ​​функция прекращения совместного использования; после остановки общие элементы остаются в списке для упрощения управления.
- Добавлена ​​кнопка «Хранилище» на боковой панели «Файлы» для упрощения управления подключенным хранилищем.

### Fixed
- Исправлена ​​проблема, из-за которой процесс восстановления RAID не мог правильно выбрать диски.
- Исправлена ​​проблема, из-за которой участники не могли создавать пользователей с именем «элиф».
- Исправлен сбой синтаксического анализа в сети, когда сетевая карта имеет несколько логических имен.
- Исправлен сбой загрузки в Файлах, когда имена файлов содержат запятые (,).
- Исправлена ​​проблема, из-за которой отмена общего доступа SMB для папки также отменяла общий доступ для подпапок.
- Исправлен сбой предварительного просмотра для некоторых файлов HEIC в файлах.
- Исправлена ​​проблема с файлами сокетов и каналов в каталогах резервных копий.

### Optimized
- Оптимизированная страница управления хранилищем
- Оптимизированные сообщения об ошибках для резервного копирования.
- Оптимизированное отображение списка файлов в «Файлах»: преодолен предыдущий лимит просмотра 10 000 файлов, теперь поддерживается обычный просмотр большего количества файлов.
