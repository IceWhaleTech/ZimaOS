## [1.5.4-alpha1]

### New
- **알파 테스트: 암호화된 폴더 기능 추가(이 기능은 현재 테스트 단계이므로 사용하기 전에 데이터를 백업하십시오)**
- 향상된 SMB 관리: 공유 중지 기능이 추가되었습니다. 중지 후에도 공유 항목은 목록에 남아 있어 더 쉽게 관리할 수 있습니다.
- 마운트된 저장소를 더 쉽게 관리할 수 있도록 파일 사이드바에 저장소 버튼을 추가했습니다.

### Fixed
- RAID 복구 프로세스에서 디스크를 제대로 선택할 수 없는 문제가 해결되었습니다.
- 회원이 "elif"라는 사용자를 생성할 수 없는 문제가 해결되었습니다.
- 네트워크 카드에 여러 논리적 이름이 있는 경우 네트워크의 구문 분석 오류가 수정되었습니다.
- 파일 이름에 쉼표(,)가 포함된 경우 파일의 다운로드 실패 문제를 수정했습니다.
- 폴더에 대한 SMB 공유를 취소하면 하위 폴더에 대한 공유도 취소되는 문제가 해결되었습니다.
- 파일의 일부 HEIC 파일에 대한 미리보기 오류가 수정되었습니다.
- 백업 디렉터리의 소켓 및 파이프 파일 관련 문제가 해결되었습니다.

### Optimized
- 최적화된 스토리지 관리 페이지
- 백업에 대한 최적화된 오류 메시지
- 파일의 최적화된 파일 목록 표시: 이전의 10,000개 파일 탐색 제한을 돌파했으며 이제 더 많은 파일 탐색을 정상적으로 지원합니다.
