## [1.5.4-alpha1]

### New
- **Test alpha : ajout d'une fonctionnalité de dossier crypté (cette fonctionnalité est actuellement en phase de test, veuillez sauvegarder vos données avant de l'utiliser)**
- Gestion améliorée des PME : ajout de la fonction d'arrêt du partage ; après l'arrêt, les éléments partagés restent dans la liste pour une gestion plus facile
- Ajout du bouton Stockage dans la barre latérale Fichiers pour une gestion plus facile du stockage monté

### Fixed
- Correction d'un problème où le processus de réparation RAID ne pouvait pas sélectionner correctement les disques
- Correction d'un problème où les membres ne pouvaient pas créer des utilisateurs nommés "elif".
- Correction d'un échec d'analyse dans le réseau lorsqu'une carte réseau a plusieurs noms logiques
- Correction d'un échec de téléchargement dans les fichiers lorsque les noms de fichiers contiennent des virgules (,)
- Correction d'un problème où l'annulation du partage SMB pour un dossier annulait également les partages pour les sous-dossiers
- Correction d'un échec de prévisualisation pour certains fichiers HEIC dans Fichiers
- Correction d'un problème avec les fichiers socket et pipe dans les répertoires de sauvegarde

### Optimized
- Page de gestion du stockage optimisée
- Invites d'erreur optimisées pour les sauvegardes
- Affichage optimisé de la liste des fichiers dans Fichiers : dépasse la limite précédente de 10 000 fichiers, prend désormais en charge la navigation normale dans plus de fichiers.
