## [1.5.2-beta2]
### Fixed
- 백업 작업이 생성되지 않는 문제가 수정되었습니다.

## [1.5.2-beta1]
### New
- USB 프로토콜을 통해 UPS 장치에 대한 지원이 추가되었습니다.
- 파일의 HEIC 이미지에 대한 라이브 사진 미리보기 및 자동 재생이 추가되었습니다.
- 원클릭 애플리케이션 캐시 정리 기능 추가
- 내장 Markdown 편집기가 추가되었습니다.

### Fixed
- USB 드라이브를 시스템 디스크로 사용할 때 파일에 "ZimaOS-HD"가 비정상적으로 표시되는 문제를 수정했습니다.
- USB 드라이브에서 부팅할 때 백업에서 zima 저장소가 열리지 않는 문제를 수정했습니다.
- 디스크 유형 구문 분석 실패로 인해 저장 장치가 파일에 표시되지 않는 문제가 해결되었습니다.
- 마운트 지점 경로에 공백이 포함된 경우 디스플레이 이상 현상이 수정되었습니다.
- 공백이 포함된 디렉터리를 마이그레이션할 때 비정상적인 크기 계산이 수정되었습니다.
- docker-cli를 사용하여 프로젝트를 가져올 때 프로젝트 이름 충돌이 수정되었습니다.
- avahi-daemon 서비스가 예기치 않게 종료되는 안정성 문제가 수정되었습니다.

### Optimized
- docker run을 통해 컨테이너를 가져온 후 기본 프롬프트 메시지가 개선되었습니다.
- 최적화된 LocalStorage 인덱스 로직
- 초기 백업 모듈 인터페이스 및 작업 관리 인터페이스 최적화
- 정확성과 명확성을 높이기 위해 개선된 백업 정책 설명
- 이제 네트워크 위젯 규모가 동적으로 조정됩니다.
- 최적화된 앱스토어 추천 로직
