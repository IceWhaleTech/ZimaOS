## [1.5.1-beta1]
### Fixed
- 修復 Files 中 ZimaOS-HD 的取得資訊（get info）不夠準確的問題
- 修復子使用者密碼含特殊字元導致登入失敗的問題
- 修復 RAID 名稱含空白時已使用儲存空間顯示為 0 的問題
- 修復 ZVM 下載 Arch/Debian 映像連結失效的問題
- 修復部分設備建立 RAID 時不顯示磁碟的問題
- 修復遷移至 NTFS 格式磁碟失敗的問題
- 修復修改映像版本後映像拉取未生效的問題
- 修復 DLNA Friendly Name 拼寫錯誤

### Optimized
- 支援備份選擇根目錄
- 優化 Docker 映像拉取策略
- 優化應用設定中過長 Tag 的顯示問題
- 優化取得備份大小的介面
- 優化安裝私有映像或不存在映像時的錯誤訊息
- 優化 Files 左側欄取消星號（Star）的流程
- 優化 Files 資源回收桶取消全選的流程
- 優化拉取新版本的速度
- 優化備份來源名稱過長時的顯示問題
