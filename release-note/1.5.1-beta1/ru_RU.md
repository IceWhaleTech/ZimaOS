## [1.5.1-beta1]
### Fixed
- Исправлено неточное получение информации для ZimaOS-HD в файлах.
- Исправлена ​​ошибка входа в систему для суб-пользователей с паролями, содержащими специальные символы.
- Исправлено отображение использованного хранилища как 0, если имя RAID содержит пробелы.
- Исправлены недействительные ссылки для скачивания образов Arch/Debian в ZVM.
- Исправлены диски, которые не отображались при создании RAID на определенных устройствах.
- Исправлен сбой при миграции на диски в формате NTFS.
- Исправлено извлечение изображения, которое не вступало в силу после изменения версии изображения.
- Исправлена ​​орфографическая ошибка в понятном имени DLNA.

### Optimized
- Добавлена ​​поддержка выбора корневого каталога в резервных копиях.
- Оптимизированная стратегия извлечения образов Docker
- Улучшено отображение слишком длинных тегов в настройках приложения.
- Оптимизированный интерфейс для получения размера резервной копии.
- Расширенные сообщения об ошибках при установке частных или несуществующих изображений.
- Оптимизированный процесс снятия пометки элементов на боковой панели «Файлы».
- Оптимизированный рабочий процесс отмены выбора в корзине файлов.
- Улучшена скорость получения новых версий.
- Улучшено отображение слишком длинных имен источников резервных копий.
