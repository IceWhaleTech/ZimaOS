## [1.5.4-beta1]

### New
- **암호화 폴더 기능 추가 (이 기능은 현재 테스트 단계입니다 — 사용하기 전에 데이터를 백업하십시오)**
- SMB 관리 개선: "공유 일시 중지" 기능 추가 — 일시 중지 후에도 목록에 남아 있으며 언제든지 다시 활성화할 수 있습니다
- 마운트된 저장소를 더 쉽게 관리할 수 있도록 파일 왼쪽 사이드바에 저장소 버튼을 추가했습니다
- 파일은 이제 IPv6 및 도메인 이름을 통해 Samba 공유에 대한 연결을 지원합니다

### Fixed
- 시스템 수준 사용자 이름 처리를 통한 인증 우회 수정
- 로그인 페이지의 리디렉션 매개변수에서 열린 리디렉션 수정
- RAID 복구 및 확장 프로세스에서 디스크를 제대로 선택할 수 없는 문제 수정
- 회원 사용자가 "elif"를 포함하는 사용자 이름을 만들 수 없는 문제 수정
- 단일 NIC에 여러 논리적 이름이 있는 경우 네트워크의 구문 분석 오류 수정
- 파일에서 쉼표(,)가 포함된 파일을 다운로드할 수 없는 문제 수정
- 폴더에서 SMB 공유를 취소하면 해당 하위 폴더의 공유도 의도하지 않게 취소되는 문제 수정
- 파일에서 일부 HEIC 파일의 미리보기 실패 수정
- 백업 디렉터리에 소켓 또는 파이프 파일이 포함된 경우 문제 수정
- 대문자 "BTRFS"로 저장된 RAID 데이터베이스 레코드의 자동 마운트 실패 수정
- 시스템 디스크가 가득 찬 경우 휴지통을 사용할 수 없게 되는 문제 수정
- BTRFS에서 휴지통을 비워도 사용 가능한 공간이 즉시 회수되지 않는 문제 수정
- 설정에서 회원을 만든 후 공유 폴더를 선택할 수 없는 문제 수정
- USB 유형 시스템 디스크에 불필요한 추가 파티션이 마운트되는 문제 수정
- 백업 생성이 성공했지만 여전히 오류 메시지가 표시되는 문제 수정

### Optimized
- 스토리지 관리 페이지 최적화
- 백업 작업에 대한 오류 메시지 및 프롬프트 개선
- 파일의 파일 목록 표시 최적화: 이전 10,000개 파일 검색 제한 제거, 이제 훨씬 더 많은 파일의 원활한 검색 지원
- 디바이스 이름 수정 시 오류 처리 로직 최적화
