## [1.5.4-alpha3]

### New
- ファイルは、IPv6 およびドメイン名を介した Samba 共有への接続をサポートするようになりました。

### Fixed
- Windows で SMB 経由で暗号化フォルダーに大きなファイルをアップロードすると失敗する問題を修正しました
- Windows Officeを使用してSMB経由で暗号化フォルダー内のPPTファイルを変更すると、次回最初に開くときにエラーが発生する問題を修正しました

### Optimized
- SMB シナリオでの暗号化フォルダーのエクスペリエンスを改善しました (ロック解除前のリマインダー、新しいロック解除方法)

## [1.5.4-alpha2]

### Fixed
- ネットワークインターフェイスに論理名がない場合にネットワークで解析エラーが発生する問題を修正しました。
- 特定のランダムな書き込みにより暗号化されたファイルが破損する問題を修正しました

### Optimized
- ログインリダイレクトロジックを最適化しました。

## [1.5.4-alpha1]

### New
- **アルファ テスト: 新しく追加された暗号化フォルダー機能 (この機能は現在テスト段階です。使用する前にデータをバックアップしてください)**
- 新しい「共有の停止」オプションにより SMB 管理機能が改善されました。停止した共有はリストに残るため、管理が容易になります
- マウントされたストレージの管理を容易にするために、ファイルの左側のサイドバーに新しいストレージ ボタンを追加しました。

### Fixed
- RAID 修復および拡張プロセスでディスクを適切に選択できない問題を修正しました
- メンバーロールが「elif」という名前のユーザーを作成できない問題を修正しました
- 1 つのネットワーク インターフェイスに複数の論理名がある場合にネットワークで解析エラーが発生する問題を修正しました。
- ファイル名にカンマ（,）が含まれるファイルが正しくダウンロードできない問題を修正
- フォルダーの SMB 共有をキャンセルすると、サブフォルダーの共有もキャンセルされる問題を修正しました
- 一部の HEIC ファイルがファイルでプレビューできない問題を修正しました
- ソケットまたはパイプファイルを含むバックアップディレクトリの問題を修正しました
- データベースに「BTRFS」が大文字で記録されている場合に RAID 自動マウントが失敗する問題を修正しました。

### Optimized
- ストレージ管理ページを最適化しました
- バックアップ用の最適化されたエラー プロンプト
- ファイル リスト表示が最適化され、通常 10,000 個を超えるファイルの閲覧がサポートされるようになりました (以前の制限は 10,000 個でした)
