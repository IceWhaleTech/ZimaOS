## [1.5.4-alpha3]

### New
- Files는 이제 IPv6 및 도메인 이름을 통해 Samba 공유에 대한 연결을 지원합니다.

### Fixed
- Windows에서 SMB를 통해 암호화된 폴더에 대용량 파일을 업로드할 수 없는 문제가 해결되었습니다.
- Windows Office를 사용하여 SMB를 통해 암호화된 폴더의 PPT 파일을 수정하면 다음에 처음 열 때 오류가 발생하는 문제가 수정되었습니다.

### Optimized
- SMB 시나리오에서 암호화된 폴더 환경 개선(잠금 해제 전 알림, 새로운 잠금 해제 방법)

## [1.5.4-alpha2]

### Fixed
- 네트워크 인터페이스에 논리적 이름이 없을 때 네트워크의 구문 분석 오류가 수정되었습니다.
- 특정 무작위 쓰기로 인해 암호화된 파일이 손상되는 문제가 수정되었습니다.

### Optimized
- 로그인 리디렉션 로직을 최적화했습니다.

## [1.5.4-alpha1]

### New
- **알파 테스트: 새로 추가된 암호화 폴더 기능(이 기능은 현재 테스트 단계입니다. 사용하기 전에 데이터를 백업하십시오)**
- 새로운 "공유 중지" 옵션으로 SMB 관리 기능이 향상되었습니다. 중지된 공유는 목록에 남아 있어 쉽게 관리할 수 있습니다.
- 마운트된 저장소를 더 쉽게 관리할 수 있도록 파일 왼쪽 사이드바에 새 저장소 버튼을 추가했습니다.

### Fixed
- RAID 복구 및 확장 프로세스에서 디스크를 제대로 선택할 수 없는 문제를 수정했습니다.
- 회원 역할이 "elif"라는 사용자를 생성할 수 없는 문제가 해결되었습니다.
- 하나의 네트워크 인터페이스에 여러 논리적 이름이 있는 경우 네트워크의 구문 분석 오류를 수정했습니다.
- 이름에 쉼표(,)가 포함된 파일을 파일에서 제대로 다운로드할 수 없는 문제를 수정했습니다.
- 폴더에서 SMB 공유를 취소하면 하위 폴더 공유도 취소되는 문제가 해결되었습니다.
- 일부 HEIC 파일을 파일에서 미리 볼 수 없는 문제가 해결되었습니다.
- 소켓 또는 파이프 파일이 포함된 백업 디렉터리 문제를 해결했습니다.
- 데이터베이스가 "BTRFS"를 대문자로 기록할 때 RAID 자동 마운트가 실패하는 문제를 해결했습니다.

### Optimized
- 스토리지 관리 페이지 최적화
- 백업에 대한 최적화된 오류 메시지
- 최적화된 파일 목록 표시, 이제 일반적으로 10,000개 이상의 파일 검색을 지원합니다(이전 제한은 10,000개였습니다).
