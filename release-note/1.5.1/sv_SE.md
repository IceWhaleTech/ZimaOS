### Fixed
- Fixade felaktig "get info" för ZimaOS-HD i filer
- Fixat inloggningsfel för underanvändare med lösenord som innehåller specialtecken
- Fixat använd lagring som visas som 0 när RAID-namnet innehåller mellanslag
- Fixade ogiltiga nedladdningslänkar för Arch/Debian-bilder i ZVM
- Fixade diskar som inte visas när RAID skapades på vissa enheter
