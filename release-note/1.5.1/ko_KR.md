## [1.5.1]
### Fixed
- 파일에서 ZimaOS-HD에 대한 부정확한 '정보 가져오기'를 수정했습니다.
- 특수 문자가 포함된 비밀번호를 사용하는 하위 사용자의 로그인 실패 문제를 수정했습니다.
- RAID 이름에 공백이 포함된 경우 사용된 스토리지가 0으로 표시되는 문제를 수정했습니다.
- ZVM의 Arch/Debian 이미지에 대한 잘못된 다운로드 링크를 수정했습니다.
- 특정 장치에서 RAID를 생성할 때 디스크가 표시되지 않는 문제 수정
- NTFS로 포맷된 디스크로 마이그레이션할 때 발생하는 오류 수정
- 이미지 버전을 수정한 후 이미지 가져오기가 적용되지 않는 문제를 수정했습니다.
- DLNA 친숙한 이름의 철자 오류 수정

### Optimized
- 백업에서 루트 디렉터리 선택에 대한 지원이 추가되었습니다.
- 최적화된 Docker 이미지 풀 전략
- 앱 설정에서 지나치게 긴 태그 표시가 개선되었습니다.
- 백업 크기 검색에 최적화된 인터페이스
- 개인 이미지 또는 존재하지 않는 이미지를 설치할 때 오류 메시지가 향상되었습니다.
- 파일 사이드바에서 항목의 별표 표시를 취소하는 간소화된 프로세스
- 파일 휴지통의 간소화된 선택 취소 워크플로
- 새 버전을 가져오는 속도가 향상되었습니다.
- 지나치게 긴 백업 소스 이름 표시 개선
