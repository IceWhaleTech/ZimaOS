## [1.5.1]
### Fixed
- Se corrigió "obtener información" inexacta para ZimaOS-HD en Archivos
- Se corrigió el error de inicio de sesión para subusuarios con contraseñas que contienen caracteres especiales.
- Se corrigió el almacenamiento usado que se muestra como 0 cuando el nombre de RAID contiene espacios
- Se corrigieron enlaces de descarga no válidos para imágenes de Arch/Debian en ZVM
- Los discos fijos no se muestran al crear RAID en ciertos dispositivos
- Se solucionó el error al migrar a discos formateados en NTFS.
- Se corrigió que la extracción de imágenes no surtiera efecto después de modificar la versión de la imagen.
- Se corrigió el error de ortografía en el nombre descriptivo de DLNA.

### Optimized
- Se agregó soporte para seleccionar el directorio raíz en las copias de seguridad.
- Estrategia optimizada de extracción de imágenes de Docker
- Visualización mejorada de etiquetas demasiado largas en la configuración de la aplicación
- Interfaz optimizada para recuperar el tamaño de la copia de seguridad
- Mensajes de error mejorados al instalar imágenes privadas o inexistentes
- Proceso simplificado para quitar elementos destacados en la barra lateral de Archivos
- Flujo de trabajo de deselección optimizado en la Papelera de reciclaje de archivos
- Velocidad mejorada para extraer nuevas versiones.
- Visualización mejorada de nombres de fuentes de respaldo demasiado largos
