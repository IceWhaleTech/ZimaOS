## [1.5.1]
### Fixed
- Correction de « obtenir des informations » inexactes pour ZimaOS-HD dans les fichiers
- Correction d'un échec de connexion pour les sous-utilisateurs avec des mots de passe contenant des caractères spéciaux
- Correction du stockage utilisé affichant 0 lorsque le nom RAID contient des espaces
- Correction des liens de téléchargement invalides pour les images Arch/Debian dans ZVM
- Correction des disques ne s'affichant pas lors de la création d'un RAID sur certains appareils
- Correction d'un échec lors de la migration vers des disques au format NTFS
- Correction de l'extraction d'image ne prenant pas effet après la modification de la version de l'image
- Correction d'une erreur d'orthographe dans le nom convivial DLNA

### Optimized
- Ajout de la prise en charge de la sélection du répertoire racine dans les sauvegardes
- Stratégie d'extraction d'images Docker optimisée
- Affichage amélioré des tags trop longs dans les paramètres de l'application
- Interface optimisée pour récupérer la taille de la sauvegarde
- Messages d'erreur améliorés lors de l'installation d'images privées ou inexistantes
- Processus rationalisé pour supprimer les éléments favoris dans la barre latérale Fichiers
- Flux de travail de désélection rationalisé dans la corbeille des fichiers
- Vitesse améliorée d’extraction des nouvelles versions
- Affichage amélioré des noms de sources de sauvegarde trop longs
