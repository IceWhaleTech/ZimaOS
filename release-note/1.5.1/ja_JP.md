## [1.5.1]
### Fixed
- ファイル内の ZimaOS-HD の不正確な「情報を取得」を修正しました
- 特殊文字を含むパスワードを使用したサブユーザーのログイン失敗を修正
- RAID 名にスペースが含まれている場合に使用済みストレージが 0 として表示される問題を修正
- ZVM の Arch/Debian イメージの無効なダウンロード リンクを修正しました
- 特定のデバイスで RAID を作成するときにディスクが表示されない問題を修正
- NTFS フォーマットのディスクに移行する際の失敗を修正
- イメージバージョンを変更した後にイメージプルが有効にならない問題を修正
- DLNA フレンドリ名のスペルミスを修正

### Optimized
- バックアップでのルート ディレクトリの選択のサポートを追加しました
- 最適化された Docker イメージのプル戦略
- アプリ設定で長すぎるタグの表示を改善しました
- バックアップ サイズを取得するための最適化されたインターフェイス
- プライベートイメージまたは存在しないイメージをインストールするときのエラーメッセージの強化
- ファイルサイドバーでアイテムのスターを外すための合理化されたプロセス
- ファイルのごみ箱での合理化された選択解除ワークフロー
- 新しいバージョンをプルする速度の向上
- 長すぎるバックアップ ソース名の表示を改善しました。
