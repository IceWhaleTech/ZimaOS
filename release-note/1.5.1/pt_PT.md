## [1.5.1]
### Fixed
- Corrigido `get info` impreciso para ZimaOS-HD em arquivos
- Corrigida falha de login para subusuários com senhas contendo caracteres especiais
- Corrigido armazenamento usado exibido como 0 quando o nome do RAID contém espaços
- Corrigidos links de download inválidos para imagens Arch/Debian no ZVM
- Discos corrigidos que não eram exibidos ao criar RAID em determinados dispositivos
- Falha corrigida ao migrar para discos formatados em NTFS
- Corrigido o pull de imagem que não entrava em vigor após a modificação da versão da imagem
- Erro de ortografia corrigido no nome amigável do DLNA

### Optimized
- Adicionado suporte para seleção de diretório raiz em backups
- Estratégia otimizada de extração de imagem Docker
- Exibição aprimorada de tags excessivamente longas nas configurações do aplicativo
- Interface otimizada para recuperar o tamanho do backup
- Mensagens de erro aprimoradas ao instalar imagens privadas ou inexistentes
- Processo simplificado para remover itens com estrela na barra lateral de Arquivos
- Fluxo de trabalho de desmarcação simplificado na Lixeira de Arquivos
- Maior velocidade de extração de novas versões
- Exibição aprimorada de nomes de fontes de backup excessivamente longos
