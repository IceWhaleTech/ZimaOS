## [1.5.4]

### New
- **新增加密文件夹功能（此功能目前处于测试阶段，使用前请提前备份好数据）**
- 完善 SMB 管理功能，新增暂停共享功能，暂停后共享项仍保留在列表中便于管理
- Files 左侧栏新增 Storage 按钮，便于管理挂载存储
- Files 新增支持通过 IPv6 和域名连接 Samba 共享

### Fixed
- 修复通过系统级用户名处理的身份验证绕过问题
- 修复登录页面重定向参数中的开放重定向问题
- 修复 RAID 修复和扩容流程无法正常选择磁盘的问题
- 修复 Member 用户无法创建包含 "elif" 的用户名的问题
- 修复 Network 中单个网卡存在多个逻辑名时解析失败的问题
- 修复 Files 中无法下载文件名包含逗号（,）的文件的问题
- 修复取消文件夹的 SMB 共享时也会意外取消其子文件夹共享的问题
- 修复 Files 中部分 HEIC 文件预览失败的问题
- 修复备份目录包含套接字或管道文件时的问题
- 修复 RAID 数据库记录存储为大写 "BTRFS" 时自动挂载失败的问题
- 修复系统磁盘已满时回收站变得不可用的问题
- 修复在 BTRFS 上清空回收站不能立即回收可用空间的问题
- 修复在设置中创建 Member 后无法选择共享文件夹的问题
- 修复 USB 类型系统磁盘上挂载了不必要的额外分区的问题
- 修复备份创建成功但仍显示错误消息的问题

### Optimized
- 优化存储管理页面
- 改进备份操作的错误消息和提示
- 优化 Files 中的文件列表显示：移除了之前的 10,000 个文件浏览限制，现在支持流畅浏览更多数量的文件
- 优化修改设备名称时的错误处理逻辑
