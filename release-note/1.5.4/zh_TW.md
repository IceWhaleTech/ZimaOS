## [1.5.4]

### New
- **新增加密資料夾功能（這功能目前還在測試階段，使用前請先備份好資料喔）**
- 完善 SMB 管理功能，新增「暫停分享」功能，暫停後分享項目還會留在列表方便管理
- Files 左側欄新增 Storage 按鈕，方便管理掛載的儲存空間
- Files 新增支援用 IPv6 佮域名連線 Samba 分享

### Fixed
- 修好通過系統級使用者名稱處理的身份驗證繞過問題
- 修好登入頁面重新導向參數中的開放重新導向問題
- 修好 RAID 修復佮擴容流程無法正常選硬碟的問題
- 修好 Member 使用者無法建立包含 "elif" 的使用者名稱的問題
- 修好 Network 內單一網卡有好幾個邏輯名稱時解析失敗的問題
- 修好 Files 內無法下載檔名有逗號（,）的檔案的問題
- 修好取消資料夾的 SMB 分享時也會意外取消子資料夾分享的問題
- 修好 Files 內部分 HEIC 檔案預覽失敗的問題
- 修好備份目錄包含 socket 或 pipe 檔案時的問題
- 修好 RAID 資料庫記錄存成大寫 "BTRFS" 時自動掛載失敗的問題
- 修好系統磁碟已滿時資源回收筒變得不可用的問題
- 修好在 BTRFS 上清空資源回收筒無法立即回收可用空間的問題
- 修好在設定中建立 Member 後無法選擇分享資料夾的問題
- 修好 USB 類型系統磁碟上掛載了不必要的額外分區的問題
- 修好備份建立成功但仍顯示錯誤訊息的問題

### Optimized
- 優化儲存管理頁面
- 改進備份操作的錯誤訊息佮提示
- 優化 Files 內的檔案列表顯示：移除了之前的 10,000 個檔案瀏覽限制，現在支援流暢瀏覽更多數量的檔案
- 優化修改裝置名稱時的錯誤處理邏輯
